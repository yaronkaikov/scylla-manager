Version
-------

The version command shows the currently installed sctool version and the Scylla Manager server version.

version
=======

**Syntax:**

.. code-block:: none

   sctool version [global flags]

version parameters
..................

version takes the :ref:`global-flags`.

Example: version
................

.. code-block:: none

   sctool version
   Client version: 2.1-0.20200401.ce91f2ad
   Server version: 2.1-0.20200401.ce91f2ad
